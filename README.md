# 📊 ML & DL Capstone Projects

This repository contains a collection of end-to-end Machine Learning and Deep Learning capstone projects focused on solving real-world problems.

## 🚀 Projects Included
- 🚗 Autonomous Driving (Object Detection & Safety Analysis)
- 🏛️ Heritage Preservation & Tourism Analysis
- 📈 Sales Forecasting (Time Series)

Each project includes detailed implementation, analysis, and results in its respective repository.
